# CC-BY-SA-4.0 attribution for IP2Location LITE DB23

Mandatory delivery matrix. The exact string must appear in all three surfaces — none is optional. Research anchor: doc 28 §Gap 1 lines 17, 60–61, 330.

## Required string (exact, no paraphrase)

> This site or product includes IP2Location LITE data available from https://lite.ip2location.com.

Source: [IP2Location attribution policy](https://lite.ip2location.com/attribution). CC-BY-SA-4.0 §3(a)(1) requires attribution in a "reasonable, user-visible manner". `--license` CLI flag alone does **not** satisfy this; dashboard footer is the canonical IP2Location-recommended placement.

## Surface 1 — `LICENSE-third-party.md`

Repo root. Machine-grep-checkable via Semgrep rule `geoip-attribution-string-present` and CI job `attribution`. Shape:

```markdown
# Third-Party Licenses

Statnive-live bundles, links to, or depends on data from the following
third-party sources. Each source is listed with its license.

## Runtime dependencies

(Generated by `go-licenses report ./...` — see CI `license-audit` job.
All entries are MIT, Apache-2.0, BSD-2-Clause, BSD-3-Clause, or ISC.)

## Data files (non-linked)

### IP2Location LITE DB23

This site or product includes IP2Location LITE data available from
https://lite.ip2location.com.

- License: CC-BY-SA-4.0
- Source: https://lite.ip2location.com
- Role: IP → geography enrichment at `/track` ingest time (see
  [`internal/enrich/geoip.go`](internal/enrich/geoip.go))
- Update cadence: monthly, operator-driven (SCP + SIGHUP; see
  [`.claude/skills/iranian-dc-deploy/references/nsd-config.md`](.claude/skills/iranian-dc-deploy/references/nsd-config.md))
- Attribution surface: this file, `/about` JSON endpoint, and dashboard
  footer.
```

## Surface 2 — `/about` JSON endpoint

Served by the Go binary on the public HTTP surface. Schema:

```json
{
  "name": "statnive-live",
  "version": "0.8.0",
  "license": "Proprietary",
  "attributions": [
    {
      "component": "IP2Location LITE DB23",
      "license": "CC-BY-SA-4.0",
      "url": "https://lite.ip2location.com",
      "text": "This site or product includes IP2Location LITE data available from https://lite.ip2location.com."
    }
  ]
}
```

Implementation in `internal/about/about.go`:

```go
package about

const IP2LocationAttribution = "This site or product includes IP2Location LITE data available from https://lite.ip2location.com."

type Attribution struct {
    Component string `json:"component"`
    License   string `json:"license"`
    URL       string `json:"url"`
    Text      string `json:"text"`
}

var attributions = []Attribution{
    {
        Component: "IP2Location LITE DB23",
        License:   "CC-BY-SA-4.0",
        URL:       "https://lite.ip2location.com",
        Text:      IP2LocationAttribution,
    },
}
```

## Surface 3 — dashboard footer

User-visible Preact component. Renders in every dashboard view. Shape:

```tsx
// web/dashboard/components/AttributionFooter.tsx
export function AttributionFooter() {
  return (
    <footer class="attribution-footer">
      <small>
        This site or product includes IP2Location LITE data available from{' '}
        <a href="https://lite.ip2location.com" target="_blank" rel="noopener">
          https://lite.ip2location.com
        </a>.
      </small>
    </footer>
  );
}
```

Must be present on **every** dashboard route, not just `/about`. CSS ensures it is visible (no `display: none`, no `visibility: hidden`, no off-screen positioning — Semgrep rule `geoip-attribution-string-present` fires on `web/dashboard/footer.*` but the visibility audit is a manual review item at each release.

## Paid DB23 Site License — attribution waived

If leadership budgets the paid IP2Location DB23 Site License at Phase 10 (SamplePlatform cutover), attribution is waived per IP2Location commercial terms. The three surfaces become optional. The skill's Semgrep rule `geoip-attribution-string-present` is gated by a build tag `licensed_db23` — when present, the rule does not fire.

```go
// +build licensed_db23

package about

var attributions = []Attribution{}  // paid license: no attribution needed
```

(Author a second build of the binary at that point; the Semgrep rule skip is automatic.)

## Checking the delivery

Manual checklist before each release:

```bash
# Surface 1
grep -q 'This site or product includes IP2Location LITE data available from https://lite.ip2location.com' LICENSE-third-party.md
echo $?  # must be 0

# Surface 2
curl -sf http://localhost:8080/about | jq -e '.attributions[] | select(.component == "IP2Location LITE DB23") | .text' \
  | grep -q 'includes IP2Location LITE'
echo $?  # must be 0

# Surface 3
curl -sf http://localhost:8080/ | grep -c 'IP2Location LITE'
# must be ≥ 1 (footer rendered)
```

CI job `attribution` in `.github/workflows/geoip-gate.yml` automates the first two checks. Surface 3 is covered by the Playwright E2E on the dashboard (assert footer text present + link functional).
